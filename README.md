## 📝 Описание проекта
Личное портфолио, демонстрирующее навыки в веб-разработке. Сайт состоит из 6 секций:
- Главная (с неоновой анимацией и снегом)
- Навыки (анимация гирлянды и елок)
- Опыт (карточки с подсветкой)
- Образование (таймлайн и 3D-фигуры и переливание карточек при наведении)
- Контакты (форма и контактные данные)
- игры

## Добавление мини-игр
